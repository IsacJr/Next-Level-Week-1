


# Next Level Week

Projeto **Ecoleta** desenvolvido durante a next level week da RocketSeat. O objetivo da solução é facilitar a localização de pontos de coletas de materias recicláveis presentes em uma cidade. 
Este repositório contém um conjunto de 3 módulos pertencentes ao projeto, são eles: server, web e mobile.

## Começando

As orientações a seguir permitirão a montagem de um ambiente necessário para a execução do código em ambiente de desenvolvimento

### Pré-requisitos

- [Node.js](https://nodejs.org/en/)
- [React](https://reactjs.org)
- [React Native](https://facebook.github.io/react-native/)

### Instalação
Para ter uma cópia disponível desse projeto, utilize o comando:
```
https://github.com/IsacJr/Next-Level-Week-1.git
```
#### Backend

Para baixar as dependências do backend e executa-lo, utilize os seguintes comandos:
 ```
cd server
npm install
npm run dev
```

#### Frontend

Para baixar as dependências do frontend web e executa-lo, utilize os seguintes comandos:
 ```
cd web
npm install
npm run start
```

#### Mobile

Para baixar as dependências do frontend web e executa-lo, utilize os seguintes comandos:
 ```
cd mobile
npm install
npx react-native run-android
```
